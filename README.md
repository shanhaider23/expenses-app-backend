# Expenses-sharing-app-
Expenses Sharing app between friends on trip
> A React based web application for splitting expenses spent


## Tech Stack

**Front End**

- React
- HTML
- SASS
- Bootstrap

[**Back End**]

- Node
- Express
- MongoDB

## Dependencies

1. [Node Js]

## Getting Started

1. Install dependencies

```
npm install
```

2. Launch the development server

```
npm start
```
